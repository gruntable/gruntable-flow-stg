# Gruntable Code Node Reference

## Overview
The Code node executes JavaScript with access to Gruntable tables and external APIs.
Add tables in **Table Source** above, then reference them with \`$table('Name')\`.

## API Reference

### $table('TableName')
Access a table configured in Table Source.
- **Input:** Table name (case-sensitive)
- **Returns:** \`{ columns: string[], rows: Record<string,any>[] }\` or \`null\` if not found

### Return Values
Your code MUST return an object with at least one of these:

#### $table — Write to Table Output
\`\`\`javascript
return {
  $table: {
    columns: ['Column1', 'Column2'],
    rows: [
      { Column1: 'value1', Column2: 'value2' },
      { Column1: 'value3', Column2: 'value4' }
    ]
  }
};
\`\`\`

#### $node_response — Display in Run Tab
\`\`\`javascript
return {
  $node_response: {
    status: 'ok',
    message: 'Processed 42 rows',
    data: { any: 'json', structure: 'here' }
  }
};
\`\`\`

#### Combined Output
\`\`\`javascript
return {
  $table: { columns: [...], rows: [...] },
  $node_response: { status: 'ok', processed: count }
};
\`\`\`

## Rules
- Use \`async/await\` for \`fetch()\` calls
- Table names in \`$table()\` are **case-sensitive**
- Maximum **5 input tables**
- Table output only saves if you return \`$table\`

## Examples

### Example 1: Filter and Transform Table Rows
\`\`\`javascript
// Get source table
const transactions = $table('BCA Transactions');
if (!transactions) return { $node_response: { error: 'Table not found' } };

// Filter rows where Debit > 0
const filtered = transactions.rows.filter(row => row.Debit !== '');

// Transform data
const outputRows = filtered.map(row => ({
  date: row.Tgl,
  description: row.Keterangan,
  amount: parseFloat(row.Debit.replace(/,/g, '')) || 0,
  type: 'debit'
}));

return {
  $table: {
    columns: ['date', 'description', 'amount', 'type'],
    rows: outputRows
  },
  $node_response: {
    status: 'ok',
    processed: outputRows.length
  }
};
\`\`\`

### Example 2: HTTP Request with Table Data
\`\`\`javascript
const response = await fetch('https://fake-json-api.mock.beeceptor.com/users', {
  method: 'GET',
  headers: {
    'Accept': 'application/json'
  }
});

const result = await response.json();
const users = Array.isArray(result) ? result : [];

return {
  $table: {
    columns: ['id', 'name', 'username', 'email', 'company', 'address', 'phone', 'country'],
    rows: users.map(u => ({
      id: u.id,
      name: u.name,
      username: u.username,
      email: u.email,
      company: u.company,
      address: u.address,
      phone: u.phone,
      country: u.country
    }))
  },
  $node_response: {
    status: response.ok ? 'ok' : 'error',
    userCount: users.length
  }
};
\`\`\`

### Example 3: Aggregate and Summarize
\`\`\`javascript
const sales = $table('Sales Data');
if (!sales) return { $node_response: { error: 'Table not found' } };

const summary = sales.rows.reduce((acc, row) => {
  const category = row.Category || 'Unknown';
  acc[category] = (acc[category] || 0) + parseFloat(row.Amount || 0);
  return acc;
}, {});

const outputRows = Object.entries(summary).map(([category, total]) => ({
  category,
  total_sales: total
}));

return {
  $table: {
    columns: ['category', 'total_sales'],
    rows: outputRows
  },
  $node_response: {
    status: 'ok',
    categories: outputRows.length
  }
};
\`\`\`

### Example 4: Join Two Tables
\`\`\`javascript
const customers = $table('Customers');
const orders = $table('Orders');
if (!customers || !orders) return { $node_response: { error: 'Tables not found' } };

const customerMap = new Map(customers.rows.map(c => [c.id, c]));
const outputRows = orders.rows.map(order => {
  const customer = customerMap.get(order.customer_id) || {};
  return {
    order_id: order.id,
    customer_name: customer.name || 'Unknown',
    amount: order.amount
  };
});

return {
  $table: {
    columns: ['order_id', 'customer_name', 'amount'],
    rows: outputRows
  },
  $node_response: { status: 'ok', joined: outputRows.length }
};
\`\`\`

### Example 5: No Table Output (API Only)
\`\`\`javascript

await fetch('https://api.example.com/webhook', {

  body: JSON.stringify({ rows: data.rows })
});

return {
  $node_response: { status: 'ok', sent: data.rows.length }
};
\`\`\`
